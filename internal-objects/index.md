# Internal Objects


Internal objects are the database objects that Fivetran HVR:

- creates inside the replicated databases
- uses inside the repository database


The internal objects are created automatically in HVR. They are for reference only and cannot be altered manually.

> **Note:** 
**Internal Objects** are advanced HVR functions. They are not required for most tasks.
