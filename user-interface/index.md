# User Interface


This section describes how to use the HVR web user interface (web UI) to configure, manage, deploy, and monitor a replication process.

To get started with the web UI, refer to the [User Interface Overview](https://fivetran.com/docs/hvr6/user-interface/user-interface-overview) section.

## Hardware Requirements


The web UI requires a PC, laptop or tablet with at least 1024px screen width.

> **Important:** 
The web UI is not supported:

- on mobile phones
- in Internet Explorer 11
