# Downloading HVR 6


HVR Hub and Agent installation files are available for download on the [Account Settings ▶ Downloads](https://fivetran.com/docs/getting-started/fivetran-dashboard/account-settings#downloads) page of the Fivetran [dashboard](https://fivetran.com/docs/getting-started/fivetran-dashboard):

- 
If you have a HVR license, you can find the installation file (and release notes) download links in the [**Downloads**](https://fivetran.com/dashboard/account/downloads) tab.

- 
If you have both HVR and [HVA connector](https://fivetran.com/docs/connectors/databases/hva-connectors) licenses, you can find the installation file (and release notes) download links in the [**Downloads**](https://fivetran.com/dashboard/account/downloads) tab.

- 
If you have only a [HVA connector](https://fivetran.com/docs/connectors/databases/hva-connectors) license, you can find the download links in connector [setup form](https://fivetran.com/docs/getting-started/fivetran-dashboard/connectors#addanewconnection).





> **Important:** 
- 
If you do not see the **Downloads** tab, it means access must be enabled for you. For this, you need to create a [support ticket](https://support.fivetran.com/hc/en-us/requests/new).

- 
If you do not find a required HVR version in the **Downloads** tab, you can request for it by creating a [support ticket](https://support.fivetran.com/hc/en-us/requests/new).
